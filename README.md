## Set up

```shell
pip install pprintpp
pip install boto3
```

📖 [Flask installation](https://flask.palletsprojects.com/en/2.0.x/installation/)  

Windows:  

```shell
python.exe -m venv venv

venv\Scripts\activate

pip install Flask

set FLASK_APP=hello

flask run
```

[http://127.0.0.1:5000/](http://127.0.0.1:5000/)
